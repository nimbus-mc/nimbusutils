# nimbusutils

An optional client-side mod for playnimbus.net that enables extra features!

## How to Download

You can find stable releases on [Modrinth](https://modrinth.com/mod/nimbusutils/versions) and the [Releases](https://github.com/nimbus-mc/nimbusutils/releases) tab, for experimental builds you can find those in [Actions](https://github.com/nimbus-mc/nimbusutils/actions/workflows/build.yml), make sure you click the experimental release's commit and download the jar. 

### Nimnite Features

- Build Mode Keybinds
- Left Click Shooting
- Right Click ADS
- Toggle ADS

### Planned Nimnite Features

- Lag Compensation (gonna be a pita)
- Mod Compatability (disables incompatible mods like ImmediatelyFast)
