# nimbusutils

An optional client-side mod for playnimbus.net that enables extra features!

### Nimnite Features

- Build Mode Keybinds
- Left Click Shooting
- Right Click ADS

### Planned Nimnite Features

- Lag Compensation (gonna be a pita)
- Mod Compatability (disables incompatible mods like ImmediatelyFast)
